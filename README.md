# login com typescrpit e firebase

Sistema de login, com registro de novos usuários, autenticação pelo google e tela de usuário. 

## Tecnologias utilizadas:
<ul>
  <li>
Vite
   </li>
  <li>
Typescript
   </li>
   <li>
Styled Components
   </li>
   <li>
React Hook Form
   </li>
   <li>
Zod
   </li>
  <li>
React Router Dom
   </li>
  <li>
Firebase
   </li>
  <li>
React Icons
   </li>
</ul>
